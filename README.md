# Productivity-Dashboard

# About the Project

The Productivity Dashboard is a smart web application designed to help users stay organized, focused, and motivated throughout the day. 
It combines task management, time tracking, and motivational features into a clean and interactive interface.


#Features

1] Task Management (Add, delete, and organize tasks)
2] Focus Timer (Pomodoro-style productivity)
3] Dark Mode Toggle
4] Motivational Quotes Generator
5] Clean and responsive UI
